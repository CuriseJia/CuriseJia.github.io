abstract: Identifying sensitive areas on integrated circuits (ICs) susceptible to single-event-effects (SEE) is crucial for improving radiation-hardness. This study introduces the online multi-track Locating (OML) framework to enhance high-resolution online trajectory detection for the Hi'Beam-SEE system, which aims to localize the SEE-sensitive positions on IC in micrometer scale and real-time. We employed the reparameterization method to accelerate the inference speed, which merged branches of the locating part's backbone in the deployment scenario. Additionally, we designed an irregular convolution kernel, an attention mechanism, and a fused loss function to improve the positioning accuracy. The OML demonstrates exceptional real-time processing capabilities, achieving a positioning accuracy (PA) of 1.83 $\mu $m in processing data generated by the Hi'Beam-SEE system with 163 frames per second~(fps) per GPU.
